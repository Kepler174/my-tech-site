在开始使用 Git 之前，需要在你的计算机上安装它。Git 支持 Linux、macOS 和 Windows 等主流操作系统。

<!--truncate-->

## 在 Linux 上安装

在 Linux 上，通常通过发行版的包管理器进行安装。

*   **Debian/Ubuntu**:
    ```bash
    sudo apt install git-all
    ```
*   **Fedora**:
    ```bash
    sudo dnf install git-all
    ```

## 在 macOS 上安装

macOS 提供了多种安装方式：

1.  **Xcode Command Line Tools** (推荐)：
    在终端中尝试运行 `git --version`。如果尚未安装，系统会提示你就行安装。
2.  **Homebrew**:
    如果你安装了 Homebrew，可以运行：
    ```bash
    brew install git
    ```

## 在 Windows 上安装

Windows 用户可以通过以下方式安装：

1.  **官方安装包**：
    访问 [git-scm.com](https://git-scm.com/download/win) 下载最新的 Windows 安装程序。安装过程中建议保留默认选项，特别是关于 PATH 环境变量的配置。
2.  **winget**:
    ```powershell
    winget install --id Git.Git -e --source winget
    ```

安装完成后，打开终端（Windows 下建议使用 Git Bash 或 PowerShell），输入以下命令验证安装是否成功：

```bash
git --version
```
