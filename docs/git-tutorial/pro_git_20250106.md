安装完 Git 后的第一件事，就是设置你的用户名称和邮件地址。这是非常重要的一步，因为每一个 Git 的提交都会使用这些信息，并且它们会写入到你的每一次提交中，不可更改。

<!--truncate-->

## 配置用户信息

请打开终端，运行以下命令来配置你的全局用户名和邮箱：

```bash
git config --global user.name "Your Name"
git config --global user.email "your_email@example.com"
```

*   `--global` 选项表示这台机器上所有的 Git 仓库都会默认使用这个配置。
*   如果你需要在某个特定项目中使用不同的身份（例如工作项目与个人项目区分），可以在该项目目录下运行不带 `--global` 的命令。

## 配置文本编辑器

Git 可能会调用你系统中的默认文本编辑器（如提交信息编辑、解决合并冲突时）。如果你想使用自己熟悉的编辑器（如 VS Code），可以进行如下配置：

```bash
git config --global core.editor "code --wait"
```

## 检查配置信息

你可以使用以下命令查看当前所有的配置信息：

```bash
git config --list
```

或者检查某一项特定的配置：

```bash
git config user.name
```
