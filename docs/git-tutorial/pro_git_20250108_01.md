第一章终于啃完了。虽然都是些理论和配置，看起来没啥“干货”，且慢，这些其实才是区分“Git 搬运工”和“Git 工程师”的分水岭。

<!--truncate-->

## 我学到了啥？

1.  **Git 不只是个搬运工**：它不做增量备份，它做的是**全量快照**。这就像每次都给整个项目拍 CT，而不是只拍断了的那根骨头。
2.  **“三态”是核心**：**已提交**、**已修改**、**已暂存**。我刚开始学觉得“暂存区”是脱裤子放屁，被坑了几次后才发现它是真的香——它给了我在提交前第二次反悔（和整理代码）的机会。
3.  **配置**：名字和邮箱配好，编辑器选个顺手的（求别用 Vim 折磨自己了，除非你是大神）。
4.  **RTFM**：遇到问题 `-h` 一下，比去百度搜出来的陈年老帖靠谱。
