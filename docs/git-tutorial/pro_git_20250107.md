Git 拥有丰富的命令和参数。在使用过程中，如果我们忘记了某个命令的用法，有多种方式可以获取帮助信息。

<!--truncate-->

## 使用命令行帮助

你可以通过以下三种方式查看 Git 命令的详细手册（Manpage）：

1.  `git help <verb>`
2.  `git <verb> --help`
3.  `man git-<verb>`

例如，要查看 `config` 命令的详细帮助：

```bash
git help config
```

这些命令通常会打开系统手册页，内容非常详尽和严谨，包含了该命令的所有参数说明和使用示例。对于初学者来说，官方手册可能略显晦涩，但它是最权威的参考资料。

## 快速参考

如果你只是想快速查看某个命令有哪些可用选项，可以使用 `-h` 参数。这会输出一段简洁的帮助信息：

```bash
git add -h
```

这通常比查阅完整手册更加高效，不仅能看到常用的参数缩写，还能快速回忆起命令的语法结构。
